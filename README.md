# Snake Duel Arena

A zero-build static website for a local 1v1 Snake match:

- Player 1 uses `W`, `A`, `S`, `D`
- Player 2 uses the arrow keys
- `Space` starts the next round
- `R` resets the full match

## Deploy on Vercel

Import the GitHub repo into Vercel and deploy it as-is.

- Framework preset: `Other`
- Build command: none
- Output directory: `.`

Vercel can serve the root `index.html` directly, so no package install or build step is required.
